# Fitness Tracker App

## Purpose
A simple fitness tracking SPA that allows users to track workouts, meals, steps, and goals.

## Tech Used
- React
- Vite
- React Router
- Context API
- CSS
- External API 

## Features
- Multi-page navigation
- Global state management
- Fitness profile tracking
- Daily fitness tips from API
- Responsive mobile-first design

## Routes
- Dashboard
- Profile
- Meals
- Workouts

## Setup

npm install
npm run dev
