Fitness Tracker App
What This Is

This is my final project for class. It is a Single Page Application (SPA) made with React. The app lets users log in and track their meals and workouts. It also shows a dashboard with totals.

I built this project to practice React, routing, state management, and authentication.

What You Can Do In The App

Log in 

View a Dashboard

Add and delete meals

Add and delete workouts

View a profile page

Protected routes 

How It Works

The app uses:

React

React Router

Context API 

Vite 

When you log in, a token gets saved in local storage. If you’re not logged in, you can’t access the main pages.

Security Stuff I Added

Basic protected routes

Token saved in local storage

Simple form validation



How To Run It

Clone the repo

Run npm install

Run npm run dev

Open the local link in your browser

How To Deploy

I deployed it using Vercel.
It connects to my GitHub repo and automatically deploys when I push updates.
